Always show RA the images and let him choose before putting them into portfolio, ads, or any deliverable.

**Why:** RA rejected the initial image selection -- he wanted to see all options first and pick himself. Built an interactive HTML gallery with click-to-select + submit for this purpose.

**How to apply:** When selecting images for any visual deliverable, build a review gallery or show all options. Never assume which images to use.
