The /closeout command must always double-check with RA before finalizing.

**Why:** Context is about to disappear when the session ends. RA wants a chance to add anything that might be missed -- additional learnings, decisions, or things worth saving to memory.

**How to apply:** After scanning the conversation, present the full summary (what, decisions, memories, priorities, blockers) and explicitly ask "Anything else?" before writing any files. Never skip this step.

**Note:** Previously named /log but renamed to /closeout to avoid tab-completion collision with /login.
