---
name: Direct Links Preference
description: RA wants direct URLs when being asked to navigate somewhere, not step-by-step navigation instructions
type: feedback
---

Always send the direct URL when asking RA to go to a specific page (Google Cloud Console, Make.com, etc.) -- don't give multi-step navigation instructions like "go to X -> click Y -> click Z" when a direct link exists.

**Why:** RA prefers efficiency and doesn't want to click through menus when a link gets him there instantly. Also prefers seeing the raw URL rather than a hyperlinked title -- he wants to see where he's going.

**How to apply:** Whenever directing RA to a web page, provide the raw URL (not markdown-linked) so he can see and click it. Then explain what to do once there.
