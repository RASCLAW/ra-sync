All content generation skills (brand-bold, brand-callout, brand-collection, UGC, ad-creative) produce repetitive output because they copy a single reference prompt verbatim.

**Why:** One reference prompt = one surface/environment/treatment. Every generation looks the same. Feed looks like the same image copy-pasted with different products.

**How to apply:** Each skill needs a VARIETY BANK -- a pool of options for:
- Surfaces (concrete, leather, wood, metal, kraft paper, wet asphalt, slate, marble)
- Environments (beach, urban street, cafe, gym, office, car, jeepney, park)
- Text treatments (painted, stenciled, neon, chalk, carved, embossed, projected)
- Lighting (golden hour, harsh noon, studio soft, dramatic side, neon glow)
- Background colors (not just red -- navy, charcoal, forest green, warm brown)

The RULES stay fixed (what makes the format work). The CREATIVE EXECUTION rotates randomly from the bank. This applies to: brand-bold (all 4 layouts), brand-callout, brand-collection, UGC prompt writer, ad-creative.

Identified session 94. Plan the rewrite as a dedicated task.
