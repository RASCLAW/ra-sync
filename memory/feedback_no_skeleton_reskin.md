When RA says "do something LIKE X but for Y", don't swap Y's content into X's finished skeleton. The skeleton itself was rooted in X's identity — the scene list, beat structure, and story arc encode what X needed to sell. Adapting to Y means designing a *new* skeleton from Y's product reality.

**Why:** 2026-05-20 session — recreated the ElevenLabs Scribe v2 trailer at ~80% fidelity, then RA asked for "something like that but for DuberyMNL". First attempt was a straight reskin: same 12 scenes, same beats (iPhone hero → halo title → API code card → live call chat → benchmark bars → latency odometer → polaroid feature → CoverFlow → dolly deep-dive → multilingual dome → motion-blur brand → loop CTA), just with Dubery assets and Tagalog copy substituted in. RA called it "lazy halfbaked" — correctly. ElevenLabs needed: API code (it's a dev tool), benchmark vs other LLMs (accuracy claim), multilingual dome (global scale). DuberyMNL needs: the polarized claim sold as experience, product lineup variety, ₱499 price punch, labas-life lifestyle, real DM social proof. Different product = different story arc. Borrowed motion mechanics are fine (whip-pan, white-flash, dolly push, motion-blur extrude — those are vocabulary). Borrowing the entire scene list is content theft.

**How to apply:**
- *Borrow:* motion-graphics mechanics (transition vocabulary, animation techniques inventory).
- *Build fresh:* the scene list, beat structure, copy, asset choices, color palette, typography hierarchy.
- *Ask first:* "What does THIS product/brand need to sell, in what order, with what proof?" — then sequence scenes around those answers, not around the source's scenes.
- *Smell test:* if the new comp can be diff'd against the source by changing only strings/images/colors, it's a reskin. A genuine adaptation should have different scene count, different scene durations, different element types.
- The validated Dubery direction (session 2026-05-20) had 8 scenes not 12: POV polarized open → D-hit brand → 3-product orbital lineup → kinetic price punch → spec sheet film-reel → 2×2 lifestyle wall with dolly-in → real DM scroll → close with CTA. Reference: [[project_dubery_trailer_v1.md]].
