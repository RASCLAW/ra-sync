**Positioning link 2026-04-11 (session 104):** DuberyMNL's Messenger-first funnel is the live reference implementation for the RAS Creative SOLUTIONS offer. Everything learned here — ad → landing/DM → qualification → nurture → booking — ports directly into the service-business retainer product. See `project_positioning_locked.md`.

DuberyMNL is pivoting from Traffic campaign to a Messenger-first engagement funnel.

**Why:** 1 week of Traffic ads (P716 spend, 27.9K impressions, 507 LPV) produced zero sales. Filipino consumers in the P699 price range prefer chat-based ordering over landing page checkout. They want to ask "magkano?" and confirm COD availability before buying.

**How to apply:**
- Ad images have NO prices -- drives curiosity comments
- CTA: "Comment DUBERY and we'll DM you a P50 discount code"
- Auto-DM offers dual path: order via Messenger OR use code DUBERY50 on landing page
- Chatbot handles straightforward orders end-to-end, complex questions handoff to RA
- Landing page still used but upgraded with checkout + discount code support
- Content gen: /dubery-ad-creative (ads) + /dubery-ugc-prompt-writer (daily organic)
