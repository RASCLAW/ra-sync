Limit UNBOXING, GIFTED, and DELIVERY to max 1 per batch. All three anchor on the hero shot (box + pouch + cloth) and produce similar vibes. RA flagged this during session 126 bandits-glossy-black batch.

**Why:** The 3 package categories look repetitive side-by-side — same dark DUBERY box, same accessories, just different scene framing. Wastes generation budget on low-variety output.

**How to apply:** When building a 10-image batch, allocate 7-8 to person + product surface categories (WEARING, HOLDING, SELFIE, OUTFIT_MATCH, PRODUCT, FLATLAY), and at most 1-2 from package categories. Rotate which package category gets picked across batches.
