- **Dev** (Label_7): GitHub, Vercel, Claude, Anthropic, Namecheap, Make.com

- **Shopping** (Label_9): Lazada, Shopee, Grab

Filters auto-sort new incoming mail. Old labels (Personal/Label_1, Work/Label_3, Receipts/Label_4) exist but unused.
