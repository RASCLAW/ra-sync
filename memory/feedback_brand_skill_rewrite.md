Brand content skills (bold, callout, collection) need a rewrite with two clear zones:

**LOCKED (product fidelity) -- port from WF2 prompt-writer verbatim:**
- R2: Reference image is ONLY authority. Banned word list for frame colors, lens descriptors, materials. "as shown in the reference image."
- R3: render_notes 5-field template ending with "REFERENCE: dictated entirely by the reference image."
- R4: No lens reflection descriptions. Banned phrases list.

**OPEN (creative freedom):**
- Scene, surface, environment, lighting -- variety banks, full creative range
- Typography, text treatment, layout composition -- agent decides
- No fixed prompt templates -- agent builds prompts from rules + banks

**Why:** Previous brand skills had fixed prompt templates that got copied verbatim (repetitive output) AND used soft fidelity language ("matching the style") that let product appearance drift. WF2's strict banned-word approach works better.

**How to apply:** Rewrite brand-bold first as template, then apply pattern to callout + collection. Also need to re-apply the path updates (contents/assets/) since the rollback reverted those too.
