RA rejected multiple rounds of copy before landing on product-first direction. Pattern: abstract/emotional → "still sounds generic." Product-grounded → approved.

**Why:** The brand isn't about feelings — it's about the actual product doing something real (polarization changes what you see). Copy that gets poetic or tries to be cinematic sounds like any DTC brand. Copy that describes the product plainly and drops one science hint feels more credible.

**How to apply:**
- Lead with the product name and what it does, not the feeling it creates
- One sentence of "why it works" (e.g. "physically blocks the light waves that cause glare") is enough — don't explain further
- For series/collections: names only, let the images carry the identity
- Value prop: name the price gap explicitly (₱499 vs brands charging 10x), don't soften it
- CTA: delivery facts, not emotional close
- When copy sounds like it could appear on any brand's site → too generic, rewrite
